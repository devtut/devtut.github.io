# Nodejs History

Here we are going to discuss about the history of Node.js, version information and it's current status.

# Key events in each year

# 2009

  • 3rd March : The project was named as "node" (opens new window)
  • 1st October : First very early preview of npm, the Node package
    manager
  • 8th November : Ryan Dahl's (Creator of Node.js) Original Node.js Talk at JSConf 2009

# 2010

  • Express: A Node.js web development framework
  • Socket.io initial release
  • 28th April : Experimental Node.js Support on Heroku (opens new window)
  • 28th July : Ryan Dahl's Google Tech Talk on Node.js (opens new window)
  • 20th August : Node.js 0.2.0 released (opens new window)

# 2011

  • 31st March : Node.js Guide

  • 1st May : npm 1.0: Released (opens new window)

  • 1st May : Ryan Dahl's AMA on Reddit (opens new window)

  • 10th July : The Node Beginner Book, an introduction to Node.js, is completed (opens new window).

  • A comprehensive Node.js tutorial for beginners.

  • LinkedIn launched its completely overhauled mobile app with new features and new parts under the hood.

  • Uber Engineering Manager Curtis Chambers explains why his company completely re-engineered their application using Node.js to increase efficiency and improve the partner and customer experience.

    • # 2012

    • 30th January : Node.js creator Ryan Dahl steps away from Node’s day-to-day (opens new window)
    • 25th June : Node.js v0.8.0 [stable] is out (opens new window)
    • 20th December : Hapi, a Node.js framework (opens new window) is released

    # 2013

    • 30th April : The MEAN Stack: MongoDB, ExpressJS, AngularJS and Node.js (opens new window)
    • 17th May : How We Built eBay’s First Node.js Application (opens new window)
    • 15th November : PayPal releases Kraken, a Node.js framework (opens new window)
    • 22nd November : Node.js Memory Leak at Walmart (opens new window)
      • Eran Hammer of Wal-Mart labs came to the Node.js core team complaining of a memory leak he had been tracking down for months.

      # 2014

      • 15th January : TJ Fontaine takes over Node project (opens new window)
      • 23rd October : Node.js Advisory Board (opens new window)
        • Joyent and several members of the Node.js community announced a proposal for a Node.js Advisory Board as a next step towards a fully open governance model for the Node.js open source project.

      # 2015

      # Q1

      • 14th January : IO.js 1.0.0 (opens new window)
      • 10th Febraury : Joyent Moves to Establish Node.js Foundation (opens new window)
        • Joyent, IBM, Microsoft, PayPal, Fidelity, SAP and The Linux Foundation Join Forces to Support Node.js Community With Neutral and Open Governance

        # Q2

        • 14th April : npm Private Modules (opens new window)
        • 28th May : Node lead TJ Fontaine is stepping down and leaving Joyent (opens new window)
        • 13th May : Node.js and io.js are merging under the Node Foundation (opens new window)

        # Q3

      • 2nd August : [Trace - Node.js performance monitoring and debugging](http://trace.risingstack.com/)
          - Trace is a visualized microservice monitoring tool that gets you all the metrics you need when operating microservices.

          # Q4

          • 12th October : Node v4.2.0, first Long Term Support release (opens new window)
          • 8th December : Apigee, RisingStack and Yahoo join the Node.js Foundation (opens new window)
          • 8th & 9th December : Node Interactive (opens new window)
            • The first annual Node.js conference by the Node.js Foundation

            # 2016

            # Q1

            • 10th February : Express becomes an incubated project (opens new window)
            • 23rd March : The leftpad incident (opens new window)
            • 29th March : Google Cloud Platform joins the Node.js Foundation (opens new window)

            # Q2

            • 26th April : npm has 210.000 users (opens new window)

            # Q3

            • 18th July : CJ Silverio becomes the CTO of npm (opens new window)
            • 1st August : Trace, the Node.js debugging solution becomes generally available (opens new window)
            • 15th September : The first Node Interactive in Europe (opens new window)

            # Q4

            • 11th October : The yarn package manager got released (opens new window)
            • 18th October : Node.js 6 becomes the LTS version (opens new window)
